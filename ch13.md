# 基本自然語言處理
在資訊科學的領域中，「自然語言」（Natural Language, NL）是指人類日常溝通所使用的語言（如中文、英文）。將其引入程式開發中，主要涉及兩個截然不同的層面：開發介面與處理技術。

1. 程式語言的「自然語言化」
早期的程式語言如組合語言，充滿了晦澀難懂的指令。隨著技術演進，程式語言正朝著「人類可讀性」不斷靠攏。例如 Python 或 SQL，其語法結構大量借鑑了英文的邏輯與詞彙，使得非技術人員也能大致理解程式碼的意圖。近年來，隨著 生成式 AI（如大語言模型）的崛起，我們甚至進入了「自然語言程式化」的時代。開發者只需用日常語言描述邏輯，AI 就能將其轉譯為可執行的程式碼，極大地降低了開發門檻。

2. 自然語言處理 (NLP)
這是指程式如何「理解」與「生成」人類語言。這並非單純的字串比對，而是涉及複雜的計算語言學技術，通常包含以下階段：

- 斷詞與標記：將句子拆解為詞彙，並辨識其詞性。
- 語義分析：理解字裡行間的深層含義與意圖。
- 上下文關聯：維持對話的連貫性，如同與真人交談。
